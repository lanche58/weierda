丽彩石平面中涂漆

产品介绍/CONSTRUCTIONAL PARAMETER

丽彩石平面中涂漆是现彩石涂装平面效果配套的专用产品，是丽彩石花纹颜色不可分割的一部分，具有一定的弹性延伸率并创造平面的石材效果，花纹外观达到平面石材的装饰效果。

技术参数/CONSTRUCTIONAL PARAMETER

◆成分：硅丙树脂、钛白粉、特殊助剂和水；
◆颜色：颜色按色卡或石材样板调制；
◆耗漆量：理论值5-6kg/m2(实际耗漆量会施工工艺、要求的效果、基层表面粗糙度及施工环境而有差异)；
◆干燥时间：表干2h，实干24h(25℃,湿度≤75%)；
◆涂装方式：毛刷、辊筒、有气喷涂（建议）；
◆稀释：使用清水稀释。稀释量约10-20%（不同环境条件下，按其吸收涂料的能力，在不同涂装遍数时稀释比例需做适当调整）；
◆干膜厚度：30μm(每层)；
◆施工条件：在5℃以上及相对湿度85%以下施工；
◆清洗：所有工具用完后，应立即用水清洗干净；
◆贮存：存放在阴凉(5~40℃)干燥处、密封保存可储存1年以上；
◆包装规格：25kg(桶)

产品特性/CONSTRUCTIONAL PARAMETER

◆与威尔达丽彩石漆配套使用，创造出平面的石材效果◆
◆有一定的延伸率，以弥盖墙体细微裂缝◆
◆能长期防止霉菌和藻类的滋生◆
◆产品附着力强，有良好的防水透气性◆
