无机树脂板保温装饰板

将纤维增强硅酸钙板或纤维增强水泥加压板用有机树脂，采用特殊的工艺对孔隙进行封闭处理，与保温材料复合后做成的保温装饰板。


构造性参数/CONSTRUCTIONAL PARAMETER



◆面密度：12-22kg/m2 (保温层厚度30mm）
◆规格尺寸（mm）:1220*2440 1220*1200 1220*800 1220*600 等（可按客户需求定制）
◆抗拉强度：根据不同保温材料的强度而定
◆阻燃等级：无机改性不燃保温板A级、B1级
◆导热系数：符合保温材料的相关标准
◆安装方式：1.粘贴+锚固 2.点式扣件干挂或龙骨干挂
◆适用范围：多层、高层住宅小区、别墅及写字楼等民用建筑的外墙外保温节能装饰和旧建筑物墙面翻新改造
◆环保性能：无毒无害、无辐射、无腐蚀、无污染、绿色环保建材。
◆复合结构：多层夹心复合结构、双层复合结构


产品优势/CONSTRUCTIONAL PARAMETER

◆色彩丰富、装饰豪华

◆保温节能、耐久美观

◆施工快捷、安全可靠
